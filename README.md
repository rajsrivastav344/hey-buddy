📘 Hey Buddy – Student Learning Platform 🚀

Hey Buddy is a full-stack MERN (MongoDB, Express, React, Node.js) web application designed to help students access study materials like notes, PYQs, projects, lab files, and an AI chatbot in one place.

🌐 Live Demo
🎯 Frontend: https://hey-buddy-brown.vercel.app
⚙️ Backend API: https://hey-buddy-86by.onrender.com
✨ Features
📚 Subject-wise Notes
📝 Previous Year Questions (PYQs)
🧪 Lab Assignments
💡 Ready-made Projects
🤖 AI Chatbot Support
🔐 User Authentication (Login / Signup)
👤 Protected Dashboard
📱 Responsive UI (Mobile + Desktop)
🛠️ Tech Stack
Frontend:
React.js
React Router DOM
Axios
Bootstrap
Tailwind CSS
React Toastify
Backend:
Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication
CORS
Deployment:
Frontend: Vercel
Backend: Render
Database: MongoDB Atlas
